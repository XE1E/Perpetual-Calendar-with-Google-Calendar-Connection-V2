Perpetual Calendar

LED Calendar with Google Calendar Integration

Recommended

Standard Version

The classic perpetual calendar displaying the current month with Google Calendar events.

  • 75 WS2811 LEDs
  • Google Calendar sync
  • Holidays, anniversaries, todos
  • Auto brightness
  • OTA updates
Your browser doesn't support Web Serial. Use Chrome or Edge. Installation not allowed. Connect via HTTPS.
Extended

Color Coded Clock

Includes everything from Standard plus a color-coded clock display using LEDs 63-67.

  • All standard features
  • Color coded time display
  • Uses LEDs 63-66 for HH:MM
  • LED 67 as separator
Your browser doesn't support Web Serial. Use Chrome or Edge. Installation not allowed. Connect via HTTPS.

How to Install

  1. Connect your ESP8266 (D1 Mini or compatible) to your computer via USB.
  2. Click the Install button for your preferred version above.
  3. Select the serial port when prompted by your browser.
  4. Wait for the installation to complete (about 30 seconds).
  5. Connect to WiFi: After flashing, connect to the "PerpetualCalendar-XXXX" WiFi network (password: admin1234).
  6. Configure: Open http://192.168.4.1 in your browser to set up WiFi and Google Calendar.
Note: This requires Chrome, Edge, or Opera browser. Firefox and Safari do not support Web Serial API. Make sure no other program (like Arduino IDE) is using the serial port.
