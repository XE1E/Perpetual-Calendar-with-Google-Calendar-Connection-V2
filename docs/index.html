Calendario Perpetuo

Calendario LED con Integración de Google Calendar

Recomendado

Versión Estándar

El calendario perpetuo clásico que muestra el mes actual con eventos de Google Calendar.

  • 75 LEDs WS2811
  • Sincronización con Google Calendar
  • Festivos, aniversarios, tareas
  • Brillo automático
  • Actualizaciones OTA
Tu navegador no soporta Web Serial. Usa Chrome o Edge. Instalación no permitida. Conecta vía HTTPS.
Extendido

Reloj de Colores

Incluye todo lo de la versión Estándar más un reloj con código de colores usando los LEDs 62-67.

  • Todas las funciones estándar
  • Muestra HH:MM:SS con colores
  • LEDs 62-63 para horas
  • LEDs 64-65 para minutos, 66-67 para segundos
Tu navegador no soporta Web Serial. Usa Chrome o Edge. Instalación no permitida. Conecta vía HTTPS.

Cómo Instalar

  1. Conecta tu ESP8266 (D1 Mini o compatible) a tu computadora vía USB.
  2. Haz clic en el botón Instalar de la versión que prefieras arriba.
  3. Selecciona el puerto serial cuando tu navegador lo solicite.
  4. Espera a que la instalación se complete (aproximadamente 30 segundos).
  5. Conéctate al WiFi: Después de flashear, conéctate a la red WiFi "PerpetualCalendar-XXXX" (contraseña: admin1234).
  6. Configura: Abre http://192.168.4.1 en tu navegador para configurar WiFi y Google Calendar.
Nota: Esto requiere navegador Chrome, Edge u Opera. Firefox y Safari no soportan la API Web Serial. Asegúrate de que ningún otro programa (como Arduino IDE) esté usando el puerto serial.
