# Capturas de Pantalla — Guía de Google Calendar

Esta carpeta contiene las capturas usadas en la [Guía de Configuración de Google Calendar](../../../GOOGLE_CALENDAR_SETUP.md).

Las guías (ES y EN) incluyen **marcadores** (líneas que empiezan con 📷) donde irá cada imagen. Para activarla:

1. Captura la pantalla indicada.
2. Guárdala en esta carpeta con el nombre sugerido.
3. En la guía, cambia la línea del marcador por:
   `![descripción](docs/images/gcal/nombre.png)` (en ambos archivos, `.md` y `.en.md`).
